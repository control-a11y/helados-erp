# Helados y Más — ERP (build público)

Aplicación web compilada del ERP "Helados y Más" (Catálogo, Proveedores, Compras…),
publicada para revisión vía GitHub Pages. **Este repo contiene solo el build**; el
código fuente vive en un repositorio privado.

- Requiere iniciar sesión; los datos están protegidos por autenticación + RLS multiempresa.
- La clave incluida es la *publishable* de Supabase (diseñada para clientes web).
